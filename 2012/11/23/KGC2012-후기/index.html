KGC2012 후기

2012-11-23
  • KGC2012
  • 후기

3 줄 요약.

  1. 일부 내용은 이미 알고 있던 내용들입니다만, 새로 알게 된 내용들도 있었습니다.
  2. Windows 8 스타일 게임의 특징은 터치 위주의 UX를 가지고 있고, … 체 화면으로 동작한다는 점, 다른 앱과의 상호작용이 가능하다는 점입니다.
  3. 게임 프로그래밍 가능한 언어와 기술로는 XAML, DirectX, Javascript, C#, C++/CX 등이 있고요.
KGC 2012는 10월달에 있었는데 페이스북에는 정리를 했었는데 찾기가 너무 힘들더라고요. 그래서 블로그에 옮겨 적습니다. 사진이나 그림이 없는 점은 안습. 시간이 나면 더 보기 좋게 편집해볼 생각입니다.





여러분께 KGC2012에서 제가 들은 강연에 대해서 설명해드리겠습니다 :) 일단 먼저, 여기에 작성할 때 강연 내용에 대한 순서는 꽤 뒤죽

…생략…

경우에 리젝 사유가 된다고 하네요. 예를 들면 가속도 센서만 이용해서 즐길 수 있는 게임의 경우 데스크탑이나 노트북에서는 즐길 수가 없으니까요.제가 정리한 것은 여기까지 입니다.
